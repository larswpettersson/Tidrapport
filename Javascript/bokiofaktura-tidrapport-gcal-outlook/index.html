Minimal frontend-MVP i en fil. För säker publik demo: använd egen Apps Script deployment med token i Script Properties, och skapa fakturautkast utan att skicka token i URL.

1) API och auth

Setup-guide (säker publik demo): varje testare använder egen Apps Script deployment.

  1. Deploya egen Code.gs som Web App och kopiera /exec-URL.
  2. Sätt Script Property BOKIO_API_TOKEN i Apps Script.
  3. Låt ALLOW_TOKEN_FROM_REQUEST vara tom/false.
  4. Klistra in din egen /exec i fältet Bokio API base nedan och ersätt den förifyllda standard-URL:en.

Rekommenderat: spara BOKIO_API_TOKEN i Script Properties pa din egen Apps Script deployment. Da skickas ingen token fran browsern.
